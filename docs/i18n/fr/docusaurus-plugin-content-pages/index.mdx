**EvalAP** (**Eval**uation **A**PI et **P**lateforme) est un service de haut niveau conçu pour effectuer des évaluations de modèles. 
Il fournit une API complète conçue pour les Grands Modèles de Langage (LLM) et les agents IA en général. Il propose également une plateforme pour naviguer, analyser et partager les résultats.
